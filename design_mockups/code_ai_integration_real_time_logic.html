# Academic Atelier: AI Integration & Real-time Quiz Logic This document covers the implementation of the **Interactive AI Lessons** and the **Real-time Quiz Engine** using OpenRouter and Socket.io. ## 1. OpenRouter AI Lesson Generation The lesson generator uses the `nvidia/nemotron-3-super-120b-a12b:free` model to transform teacher inputs into a structured JSON array of lesson blocks. ### AI Generation Controller `backend/controllers/lessonController.js` ```javascript const axios = require('axios'); const { PrismaClient } = require('@prisma/client'); const prisma = new PrismaClient(); exports.generateAILesson = async (req, res) => { const { title, topic, objectives, difficulty, duration } = req.body; const apiKey = process.env.OPENROUTER_API_KEY; const prompt = ` Generate a structured, block-based interactive lesson for: Title: ${title} Topic: ${topic} Objectives: ${objectives.join(', ')} Difficulty: ${difficulty} Target Duration: ${duration} minutes Format the output as a JSON array of blocks. Each block must have: - type: "text" | "code" | "quiz" | "poll" | "discussion" - content: { header: string, body: string (markdown support), code: string (for code blocks), language: string, options: string[] (for quiz/poll), correctAnswer: any } Ensure the lesson is pedagogical, starting with concepts and moving to interaction. `; try { const response = await axios.post('https://openrouter.ai/api/v1/chat/completions', { model: "nvidia/nemotron-3-super-120b-a12b:free", messages: [{ role: "user", content: prompt }], response_format: { type: "json_object" } }, { headers: { "Authorization": `Bearer ${apiKey}`, "Content-Type": "application/json" } }); const aiBlocks = JSON.parse(response.data.choices[0].message.content).blocks; // Save as Draft const lesson = await prisma.lesson.create({ data: { title, topic, objectives, classId: req.body.classId, blocks: { create: aiBlocks.map((block, index) => ({ type: block.type, content: block.content, orderIndex: index })) } }, include: { blocks: true } }); res.status(201).json(lesson); } catch (error) { res.status(500).json({ message: "AI Generation failed", error: error.message }); } }; ``` ## 2. Real-time Quiz Sync (Socket.io) Handles the "Kahoot-style" synchronized experience where the teacher controls the flow. ### Socket Utility `backend/utils/socketHandler.js` ```javascript module.exports = (io) => { io.on('connection', (socket) => { console.log('User connected:', socket.id); // Join lesson room socket.on('join_lesson', (lessonId) => { socket.join(`lesson_${lessonId}`); }); // Teacher starts quiz block socket.on('start_quiz', async ({ lessonId, blockId }) => { io.to(`lesson_${lessonId}`).emit('quiz_started', { blockId }); }); // Student submits answer socket.on('submit_answer', async ({ lessonId, sessionId, studentId, answer }) => { // Logic to save response and calculate live leaderboard const leaderboard = await calculateLeaderboard(sessionId); io.to(`lesson_${lessonId}`).emit('update_leaderboard', leaderboard); }); // Teacher moves to next block socket.on('next_block', ({ lessonId, nextIndex }) => { io.to(`lesson_${lessonId}`).emit('sync_navigation', { nextIndex }); }); }); }; ``` ## 3. Student AI Assistant (Context-Aware) The chatbot side-panel provides hints without giving direct answers. ### AI Assistant Route `backend/routes/chatRoutes.js` ```javascript router.post('/lesson-chat/message', async (req, res) => { const { lessonId, message, lessonContext } = req.body; const systemPrompt = ` You are the Academic Atelier Assistant. Context: ${lessonContext} Rule: Help the student understand concepts. Provide hints and scaffolding. STRICT RULE: Never provide the direct answer to a quiz or exercise. `; const response = await axios.post('https://openrouter.ai/api/v1/chat/completions', { model: "nvidia/nemotron-3-super-120b-a12b:free", messages: [ { role: "system", content: systemPrompt }, { role: "user", content: message } ] }, { headers: { "Authorization": `Bearer ${process.env.OPENROUTER_API_KEY}` } }); res.json({ reply: response.data.choices[0].message.content }); }); ```
